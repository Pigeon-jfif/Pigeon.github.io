Plastic Pizzas - redesign retro Japanese VHS/cassette

File inclusi:
- index.html
- style.css
- app.js
- Dischi.csv

Come provarlo:
1. Apri questa cartella in VS Code o terminale.
2. Avvia un server locale, ad esempio: python3 -m http.server 8000
3. Vai su http://localhost:8000

Nota: se apri index.html con doppio clic, alcuni browser bloccano la lettura automatica del CSV. Il sito include comunque il fallback "Scegli CSV".
